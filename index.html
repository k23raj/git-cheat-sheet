Git Cheat Sheet

Commands Desc
git config --global user.name Kamaraj that will be attached to your commits
git config --global user.email Email that will be attached to your commits
git init Intialize an empty directory
git clone [url] Email that will be attached to your commits
git status Show modified files
git add [file] Add files to staging area
git diff [file] Show changes b/w working dir and staging
git commit Commit your staged content as a new commit snapshot
git branch [branch_name] Create new branch, refering current head
git checkout [-b] [branch_name] Switching work dir. ( -b : create new branch if it doesn't exist )
git merge [branch_name] Join specified [branch] into current branch
git log List commit history
git remote add [name] [url] Create a new connection to a remote repo
git pull [remote] Fetch from remote & merge local current branch with its upstream
git push -u [remote] [branch]

(e.g) git push -u origin main

 Push local branch to remote repo
